## EAS Build preview build

```cmd
eas build -p android --profile=preview
```

## Start Emulator

```cmd
emulator -avd Pixel_7_Pro
```

## Install the latest EAS CLI

```cmd
npm install -g eas-cli
```

## Build Local (macOS, linux)

eas build --platform android --local
